# Wind Energy Project Finance Model

## Overview
This project analyzes the financial feasibility of an 80 MW wind energy project using project finance techniques. The model evaluates construction costs, operating expenses, financing structure, and long-term revenue from electricity sales.

## Project Structure
The analysis includes several key project finance components:

- Engineering, Procurement, and Construction (EPC) contract
- Long-term power purchase agreement (PPA)
- Operating and maintenance contract
- Land lease agreement
- Construction and take-out debt financing

## Financial Analysis
The project evaluates financial performance using:

- Levered Internal Rate of Return (IRR)
- Net Present Value (NPV)
- Debt Service Coverage Ratio (DSCR)
- Long-term cash flow projections

## Results
The financial model estimates an equity IRR of approximately 15% and a positive NPV under base-case assumptions, suggesting the project is financially viable.
